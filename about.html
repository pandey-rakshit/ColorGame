Summary

Well this game is really simple, build using HTML, Css and javascript.
The main purpose of this game is to guess Correct RGB(red, green, blue) color content in color samples..
ie., A RGB Color Content is provided, and you need to guess the correct box which is made with the same color content provided to you.
